Projects


HikeHarvard is an application designed for backpackers in the New England area. It helps users find hiking trips and provides them with directions from their location to the trailhead. The application also enables users to enter descriptions and comments on trips ranging from day-hikes to weeklong ventures in the woods.


FOP Leader Web Toolkit

Harvard's First-Year Outdoor Program (FOP) is an organization that serves more than a quarter of Harvard's incoming freshman class each year. The FOP Leader Web Toolkit is a web application constructed for FOP that resolves the program's paper-based data collection and storage methods. The application allows leaders to locate information about past trips and store their own data for future leaders upon returning to campus.
